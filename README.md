# TA - Return of the Pokedex

TA Pokedex created in Vue.js.

## Project setup

```
npm install
```

## Developer

* Compiles and hot-reloads for development
```
npm run serve

* Compiles and minifies for production
```
npm run build

* Run your tests
```
npm run test

* Lints and fixes files
```
